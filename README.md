# Pomodoro + Todo

Combined Pomodoro timer and todo list for maximum productivity.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Start Working** (or Skip).

**Step 3:** In the right panel, type a task and click **Add** (or press Enter). Build your to-do list.

**Step 4:** In the left panel, click the Play button to start a 25-minute focus timer.

**Step 5:** Work on your tasks while the timer counts down.

**Step 6:** Click the circle next to a task to mark it as done. The text gets crossed out.

**Step 7:** When the timer ends, the session counter increases. Switch to Short Break (5min) or Long Break (15min).

**Step 8:** Click Skip to jump to the next timer mode (Focus, Short Break, Long Break).

**Step 9:** Your tasks are saved automatically. Close and reopen the browser to continue where you left off.

## Features
- Pomodoro timer (25/5/15 min modes)
- Todo list with add/done/delete
- Session counter
- Task progress tracking (Done vs Remaining)
- LocalStorage persistence for both timer and tasks
- Side-by-side layout
